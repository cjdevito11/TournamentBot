# data/help/help.md
Welcome to D2 Hustlers Tournament Bot.

Quick start:
- `/event info <event_id>` to see status
- `/event join <event_id>` to register
- `/event bracket_image <event_id>` to view the bracket
- `/event report <event_id> <match_code> <winner_seed>` to report results

For the full list, run `/commands`.
